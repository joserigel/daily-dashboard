# Daily Dashboard
Electron app covering my daily usage of expenses, university canteen, schedules and todo list.

### Running Frontend
To run the frontend app for development, run
```
npm start
```
in the frontend directory

To run the backend for development, run
```
mvn spring-boot:run
```
in the backend directory
